# 방구석 라이브

이 프로젝트는 아주대학교 캡스톤 디자인에서 진행했던 안드로이드 앱 프로젝트이다.

앱의 이름은 '방구석 라이브' 이며 이 앱은  두명의 유저가 스트리밍을 통해 노래대결을 펼치는 앱이다.
